# Changelog

All notable changes to MinerToggle+ will be documented here.

## [1.0.0] - 2025

### Added
- **Vein Miner** — Break entire ore veins at once with sneak activation
- **Tree Capitator** — Chop full trees in one swing; auto-clears leaves
- **Auto Smelt** — Instant ore smelting on break, Fortune compatible
  - Iron, Gold, Copper, Nether Gold, Ancient Debris supported
  - Awards furnace-equivalent XP
- **Auto Replant** — Automatically replants crops on harvest
  - Wheat, Carrot, Potato, Beetroot, Nether Wart, Cocoa, Pitcher, Torchflower
- **Silk Spawners** — Mine spawners with Silk Touch
- **Double Drops** — Configurable chance to double ore drops
- **Auto Pickup** — Instant inventory collection; overflow drops at feet
- **Cave Vision** — Silent Night Vision underground (below Y=64)
  - Overworld-only mode configurable
  - No icon, no particles — completely invisible effect
- **Auto Torch** — Auto-places torches in dark areas using player's inventory
- **No Fall Damage** — Cancels fall damage for the player
- **Ore Radar** — Actionbar notifications for nearby rare ores
  - Configurable ore list, scan radius and scan interval
- **XP Boost** — Configurable multiplier on ore mining XP
- **ClearLagg System** — Timed entity/item clearer
  - Configurable interval, warning countdown, world filtering
  - Toggle items, hostile mobs, passive mobs independently
  - Keep named entities option
- **Global Mob Drops** — Admin toggle for all mob loot
- **Global Leaf Decay** — Admin toggle for natural leaf decay
- Full `/mt` command with tab completion
- Per-player toggle persistence for the session
- Fully configurable `config.yml`
- Permission nodes for every feature
